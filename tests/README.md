# TableLogic Unit Tests

## Prerequisites
- Node.js (v18.x or higher)
- NPM (Node Package Manager)

## Install Dependencies
To install the Jest testing framework, run the following command from the project root:
`npm install`

## Run All Tests
To execute the unit test suite, run:
`npm test`

## Run Tests with Coverage
To see how much of our code is covered by tests, run:
`npx jest --coverage`

## Test Output
Results are printed directly to the console. Screenshots of passing test suites are archived in `/docs/test-results/`.
